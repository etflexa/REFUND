# 💸 Refund - Sistema de Solicitação de Reembolso

## 📌 Sobre o projeto

O **Refund** é uma interface de sistema para solicitação e controle de reembolsos, desenvolvido com foco em organização, clareza de informações e experiência do usuário.

A aplicação permite registrar despesas, categorizá-las e visualizar um resumo com valores totais e lista de solicitações.

---

## 🎨 Tecnologias utilizadas

* HTML5
* CSS3

---

## ✨ Funcionalidades

* Cadastro de despesas
* Seleção de categoria
* Inserção de valores monetários
* Listagem de despesas cadastradas
* Exibição do total de gastos
* Interface organizada e intuitiva
* Remoção de itens da lista

---

## 🧠 Estrutura da aplicação

### 📝 Formulário de reembolso

* Campo para nome da despesa
* Seleção de categoria:

  * Alimentação
  * Hospedagem
  * Serviços
  * Transporte
  * Outros
* Campo para valor da despesa
* Botão para adicionar despesa

---

### 📊 Painel de controle

* Quantidade total de despesas
* Soma total dos valores
* Listagem detalhada contendo:

  * Ícone da categoria
  * Nome da despesa
  * Tipo/categoria
  * Valor
  * Opção de remoção
